## Descripción

- Este apartado es para usuarios un poco más avanzados, pero sirve, en pocas palabras, para crear una base de datos de nuestro servidor, esto ya varía dependiendo de las necesidades de cada usuario con su proyecto.

## Apartados:

### Secciones

| Secciones        | Descripcion                  |
| --------------- | ------------------------------ |
| `New Database` | Al dar click, se nos abre una ventana en la que podemos crear una nueva base de datos, en la que se puede dar un nombre y en el recuadro debajo establecer una especie de “Whitelist” en la que podemos indicar qué conexiones en específico queremos permitir que accedan a nuestra base de datos. Al tener nuestra base de datos, se verá reflejado en el listado con una ID y el nombre que escogimos, ahora al dar click en el botón que tenemos en la parte derecha que tiene forma de un ojo, nos mostrará una ventana en la que podremos ver toda la información de la misma. Endpoint es la dirección IP y el port a donde es la conexión de nuestra base, Connections From muestra las conexiones que queremos en nuestra base de datos; un % indica que todas las conexiones están permitidas. Username es el nombre de usuario que usaremos para iniciar sesión en PHPMyAdmin. Password es la contraseña que usaremos para iniciar sesión en PHPMyAdmin. JBDC Code String se utiliza para cargar el driver e indicar las configuraciones necesarias para poder establecer una conexión con la base de datos.  |
| `Rotate Password` | Al hacer click, nos permite crear una nueva contraseña para la base de datos. |
