## Descripción

- En esta sección podrás modificar algunos datos de tu servidor, así como ver los datos del SFTP, reinstalación, etc. Vayamos por partes.

## Apartados:

### Secciones

| Apartados        | Descripcion                  |
| --------------- | ------------------------------ |
| `SFTP Details` | En Server Address, veremos la dirección del servidor a la que nos tendremos que conectar. Username, es el nombre de usuario que tendremos que usar para podernos conectar al servidor FTP. La contraseña es exactamente la misma que usan para iniciar sesión a la Dashboard de BoxMine. Hay muchos programas para acceder al SFTP, pero recomendamos usar Filezilla ya que es muy fácil de usar, entender y usando esto, se podrán subir absolutamente TODO tipo de archivos, así como carpetas, solamente no hay que subir archivos comprimidos ya que el servidor no tiene la capacidad de descomprimirlos.|
| `Change Server Details` |En Server Name podemos modificar el nombre de nuestro servidor. Server Description nos da la opción de ponerle una descripción a nuestro servidor.. |
| `Debug Information`    | Aquí simplemente podremos ver en qué nodo y qué tipo de servidor tenemos así como la ID del mismo.  |
| `Reinstall Server`  | Aquí simplemente podremos ver en qué nodo y qué tipo de servidor tenemos así como la ID del mismo. |
