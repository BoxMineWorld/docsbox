## Descripción

- Este apartado sirve para dejar cosas programadas para que se ejecuten más tarde, tales como reiniciar el bot, instalar, eliminar, editar, etc.

## Imagen de Guía

<img src='/img/panel/schedule.jpg' alt='Guia Schedule' />

## Contenido del apartado Schedules:

### Contenido

| Apartado        | Descripcion                  |
| --------------- | ------------------------------ |
| `Create Schdeule` | Al hacer click, una ventana se abrirá, en la que podremos crear un nuevo schedule. Schedule name se le da un nombre al mismo. En la parte inferior en donde se pone un nombre, podemos poner una fecha y hora exacta en el que queremos que se ejecute este. Show Cheatsheet muestra una documentación a usar para poder configurar el schedule. Only When Server Is Online al habilitar esta opción, el schedule sólo funcionará si el servidor está encendido. Schedule Enabled esta opción permite ejecutar el schedule automáticamente.|
