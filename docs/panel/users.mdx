## New User

- Al dar click aquí, se abrirá una ventana en la que podremos, primero agregar un nuevo usuario a nuestro proyecto, debemos saber con exactitud su correo para poderlo agregar, respetando mayúsculas y minúsculas del mismo. Después puedes escoger los permisos que le darás a este usuario, si el permiso que le das está seleccionado, entonces este usuario tendrá acceso, al ser caso contrario, el usuario no tendrá permiso.
