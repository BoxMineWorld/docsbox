## Apartados:

### Principales

| Apartados        | Descripcion                  |
| --------------- | ------------------------------ |
| `CPU Load` | Esto se encarga de mostrarte el uso actual del procesador que tiene tu servidor actualmente. Recuerda que tener un bot optimizado siempre es garantía de no hacer que el procesador de tu servidor explote.        |
| `Uptime` | Este reloj (por así decirlo) indica la cantidad de tiempo exacto que tu servidor ha estado activo desde el último reinicio. |
| `Memory`    | Esto sirve para que sepas cuánta memoria RAM está usando tu servidor actualmente, recuerda no sobrecargar tu bot de muchísimas cosas o podría presentar lag, crasheos o tiempos prolongados de espera al usar un comando o función.  |
| `Disk`  | Aquí podrás saber cuál es el límite de espacio en almacenamiento que tienes en tu servidor. Una vez que llegues al límite no podrás instalar más paquetes hasta que liberes espacio o contrates un paquete con mayor capacidad.         |
| `Network`  | Esto se encarga de indicar el uso de ancho de banda que tu servidor está usando. 
Recuerda: No uses muchas cosas que necesiten mucho (ancho de banda) ya que es muy seguro que tu bot presente lag y tiempos muy prolongados de espera al usar comandos y funciones.         |

- Ahora en la parte inferior de lo ya explicado, llegamos a la parte de la consola; ese recuadro grande, es la consola de tu servidor, en ella verás todos y cada uno de los acontecimientos de tu bot o sitio web, ahí sabrás si está teniendo un buen funcionamiento, si hay advertencias, errores, etc. 
OJO: Es altamente recomendable tener un nivel intermedio de inglés ya que absolutamente todos los registros que aparecen ahí, son en dicho idioma únicamente:

### Botones

| Boton        | Descripcion                                                  |
| ------------------ | ----------------------------------------------------------- |
| `Botón Start`  | Al dar click, inicias tu bot/sitio web. |
| `Botón Restart`      | Al dar click, reinicias tu bot/sitio web.            |
| `Botón Stop` | Al dar click, detienes tu bot/sitio web.         |

- Por cuestiones de seguridad, no está permitido que envíes comandos desde tu consola en la mayoria de los casos, existen algunas excepciones con algunos sistemas o bots.
