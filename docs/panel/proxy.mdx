## Disclaimer

- Este apartado es única y exclusivamente para los que tienen un sitio web en BoxMine, si este no es tu caso, puedes omitir esta sección de la guía.

## Descripción

Aquí vamos a crear un nueva conexión para que nuestro servidor tenga un dominio y todos los usuarios se puedan conectar a nuestro sitio web.

## Apartados:

### Información

| Sección        | Descripción                  |
| --------------- | ------------------------------ |
| `Create Proxy` | Al dar click, una pequeña ventana se abrirá. En Proxy Domain deberás poner el dominio exacto al que el proxy deberá establecer conexión, si no cuentas con uno, tendrás que dirigirte a nuestro servidor en Discord y solicitar la guía para poder tener tu dominio, en caso de que ya lo tengas, introducelo en el campo, tiene que estar algo como: midominio.com o en caso de que tengas un subdominio en BoxMine tiene que ser algo como: midominio.boxmine.xyz. Allocation es la dirección IP y el PORT de nuestro servidor a donde el proxy se conectará, es recomendable que no modifiques esta opción. Enable SSL esta opción habilitará un certificado SSL a tu sitio web para que la carga siempre sea detectada como segura (https) e incremente la seguridad de tu sitio. |

- Una vez creado el proxy, tu nuevo link para tu sitio web será algo como esto: https://unnombre.com o el caso de ser un subdominio de BoxMine será algo así: https://unnombre.boxmine.xyz.
