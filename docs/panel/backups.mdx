## Descripción

- Este apartado es uno de los más importantes, en este puedes crear una copia de seguridad de tu servidor por completo.

## Apartados

### Secciones de la página

| Secciones        | Descripcion                  |
| --------------- | ------------------------------ |
| `Create Backup` |Al dar click, se abrirá una ventana en la que podremos establecer un nombre para nuestra copia, una lista de los archivos que no queremos que sean respaldados y una opción de bloquear nuestra copia de ser eliminada a menos que nosotros escojamos hacerlo. Una vez que esté todo listo, el servidor comenzará a crearla y en unos momentos estará ahí. Ahora a la derecha de nuestra backup podremos ver que hay tres puntos, al darle click, saldrá un menú y ahí mismo veremos tres opciones. |
| `Download` | Esta opción nos permite descargar nuestra copia de seguridad a nuestra computadora, el archivo se comprime en un archivo .tcz (WinRAR puede descomprimirlo). |
| `Restore`    | Esta opción realiza una restauración de la copia de seguridad, es decir, la crea desde el principio agregando posibles nuevos archivos.  |
| `Disk`  | Aquí podrás saber cuál es el límite de espacio en almacenamiento que tienes en tu servidor. Una vez que llegues al límite no podrás instalar más paquetes hasta que liberes espacio o contrates un paquete con mayor capacidad.         |
| `Lock`  | Esta opción nos permite bloquear nuestra copia de seguridad para que no sea eliminada. |

## Conclusión

- Las copias de seguridad son altamente recomendadas, ya que si el nodo requiere algún reinicio general o mantenimiento, se evita perder archivos en absoluto.
