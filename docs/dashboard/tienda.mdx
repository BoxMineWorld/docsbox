## Descripción

En esta página podrás comprar distinta mercancía exclusiva de BoxMine, desde camisetas hasta tazas con geniales diseños, a eso mismo se suma un apartado extra para adquirir BoxCoins, estos paquetes de BoxCoins son en cantidades exactas para que puedas adquirir un servidor en los nodos VIP.

## Paquetes de BoxCoins

| Paquete  | ¿Qué adquieres con este paquete? |
| --------------- | ------------------------------ |
| `Bronce` | 1,000 BoxCoins que sirven para un servidor del tipo Bronce. |
| `Silver` | 2,000 BoxCoins que sirven para un servidor del tipo Silver. O 2 servidores del tipo Bronce. |
| `Gold`    | 2,000 BoxCoins que sirven para un servidor del tipo Silver. O 2 servidores del tipo Bronce.  |

## Nota

- Los precios pueden ser un poco elevados para distintos países y sus respectivas economías, pero es una muy buena ayuda para no tener que hacer un cálculo acorde a las necesidades.
